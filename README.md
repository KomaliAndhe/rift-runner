# RIFT RUNNER

RIFT RUNNER is a browser-based 3D endless runner game inspired by Subway Surfers. Built using HTML5, CSS3, JavaScript, and Three.js, the game delivers smooth gameplay with lane switching, jumping, sliding, procedural obstacle generation, dynamic lighting, collision detection, score tracking, and responsive controls for both desktop and mobile devices.

## Features
- 3D graphics powered by Three.js
- Endless runner gameplay
- Three-lane movement system
- Jump and slide mechanics
- Procedural obstacle generation
- Coin collection system
- Dynamic speed progression
- Real-time collision detection
- Responsive keyboard and touch controls
- Pause and resume functionality
- Persistent high-score storage
- Cyberpunk-inspired UI
- Web Audio API sound effects

## Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- Three.js
- WebGL
- Web Audio API
- Local Storage API
